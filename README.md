## How to run locally:

> ⚠️ **Important:**  
> You can only run this web app locally if you have been provided with the required Supabase API keys.

### 1. Clone the repo locally.
```
git clone https://github.com/BigOh-no/Qure.git
```
### 2. Navigate to the directory called `qure`
```
cd .\qure\
```
### 3. Install the dependencies
```
npm install
```
### 4. Add in the environment variables

   - Create a .env file in the qure directory
   - Add the API keys in as instructed in the provided documentation
   - It should look something like this
  
```
REACT_APP_SUPABASE_URL= <provided URL>
REACT_APP_SUPABASE_KEY= <provided KEY>
```
### 5. Launch the app locally.
```
npm start
```
### 6. The site should deploy automatically in a browser with the following URL
```
https:\\localhost:3000
```
